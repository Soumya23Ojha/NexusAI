About NexusAI

NexusAI is a premium AI assistant interface built with HTML5, CSS3, and Vanilla JavaScript. It is designed for future integration with a Python FastAPI backend powered by the Gemini API.

  • ⚡ Fast, responsive, zero-dependency UI
  • 🎨 Premium dark/light theme with CSS variables
  • 🔌 Ready for FastAPI + Gemini API integration
  • ♿ Fully accessible and keyboard navigable
